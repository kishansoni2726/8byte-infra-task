Infra created with Terraform 


This Infrastructure Code will Creaet following Things:
    
    1.  VPC 
    2.  Internet Gateway
    3.  Public Subnet  (x3)
    4.  Private Subnet (x3)
    5.  NAT Gateway
    6.  Elastic IP
    7. Document DB Mongo Cluster


How to create Infra 


    1.  terraform init 
    2.  terraform plan
    3.  terraform apply --auto-approve
